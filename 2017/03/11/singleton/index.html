单例模式-Singleton Pattern

发表于 2017-03-11 | 分类于 设计模式 | | 阅读次数

1. 饿汉模式

优点：线程安全
缺点：无论系统在运行时是否需要使用该单例对象，在类加载时该对象就需要创建，不能实现单例对象的按需加载
